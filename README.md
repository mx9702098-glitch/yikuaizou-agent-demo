# 一块走 · Agent Demo

把「一块走」捐步流程搬进微信 Agent 对话窗口的交互 Demo。

## 在线预览

https://mx9702098-glitch.github.io/yikuaizou-agent-demo/

## 4 条路径

- ✅ **步数够了，还没捐** — 一句话 + 一个按钮完成捐步
- 🌸 **今天已经捐过了** — 查进展、提醒明日
- 🚶 **还没走够** — 不催，留直接捐+订阅两条出口
- 🔥 **连续 30 天里程碑** — 峰值体验 · 可分享朋友圈

## 本地运行

直接浏览器打开 `index.html` 即可（纯静态 HTML/CSS/JS，无依赖）。

## 技术栈

- 纯 HTML + CSS + Vanilla JS
- 模拟微信 Agent 对话流 + 卡片消息 + 原生付款 + 订阅消息授权

## 许可

仅用于腾讯公益内部 Demo 演示。
